# 基于SSH的宿舍管理系统7

#### 介绍
基于SSH的宿舍管理系统7分为系统管理员、楼宇管理员、学生三种权限

- 系统管理员：宿舍管理、学生管理、楼宇管理、缺寝管理、寝室管理、迁出迁入管理、设置楼宇公管理员等

- 楼宇管理员：迁出和缺寝记录管理、宿舍管理、消息处理、学生缺寝登记、学生入住登记等

- 学生：缺勤记录查询确认、申请宿舍调换、申请宿舍迁出等
